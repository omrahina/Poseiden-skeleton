# spring-boot
## Technical:

1. Framework: Spring Boot v2.4.5
2. Apache Maven 3.6.1
3. Java 11
4. Thymeleaf
5. Bootstrap v.4.3.1

## Run

Please run the sql commands present in the ``` data.sql ``` file under the ``` doc ``` folder to set up the tables in the database.

Build the project

```bash
mvn clean install
```
Start the application

```bash
mvn spring-boot:run
```
